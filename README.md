# Moroccan Finance Tracker

A Morocco-native personal finance tracker for salary, cash spending, budgets, and savings goals.

## MVP Goal

Help Moroccan users understand where their money goes every month using:
- MAD currency
- Moroccan spending categories
- Cash-first tracking
- Fast Darija/French transaction input
- Monthly budgets
- Savings goals
- Monthly reports

## Tech Stack

### Mobile
- Expo React Native
- TypeScript
- SQLite
- Zustand

### Backend Later
- Java
- Spring Boot
- PostgreSQL
